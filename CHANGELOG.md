# Changelog

## [0.1.0] - 2026-03-27

### Added
- UMAP embedding computation using SMILE library
- Phenotype coloring from PathClass annotations with legend
- Interactive polygon gating with UNFOCUSED classification
- Permanent population tagging via derived PathClass
- Marker expression overlay (z-score and raw intensity modes)
- Configurable subsampling (Auto/Off/Fixed) with stratified sampling
- kNN projection for non-sampled cells
- OOM protection with memory estimation and graceful recovery
- Zoom/pan on UMAP scatter plot
- CSV export of UMAP coordinates
- 34 JUnit 5 tests
